##  Grupo 3  
**NRC:** 30724

## 📖 Descripción
Este repositorio tiene como finalidad aplicar los conceptos, metodologías y herramientas propias del análisis y diseño de software, permitiendo la construcción de una solución estructurada a un problema específico dentro del ámbito tecnológico.

Se abordan fases fundamentales como el levantamiento de requisitos, modelado del sistema, diseño de arquitectura y propuesta de solución, siguiendo buenas prácticas de ingeniería de software.
